# Guía de uso rápida · celular

## 1. Inicio de sesión
- Abrir la app desde el navegador del celular.
- Iniciar con usuario y contraseña.
- Cuando haya señal, la app sincroniza registros pendientes.

## 2. Cómo usar en campo
### Registrar un animal
1. Entrar a **Registro rápido > Animal**.
2. Escribir código del arete o identificación.
3. Completar sexo, fecha, tipo de identificación y categoría.
4. Guardar.

### Registrar un peso
1. Ir a **Peso**.
2. Digitar código del animal.
3. Registrar peso y fecha.
4. Guardar.

### Registrar vacuna o tratamiento
1. Ir a **Sanidad**.
2. Digitar código del animal.
3. Seleccionar tipo de evento.
4. Registrar medicamento, dosis y fecha.
5. Guardar.

### Mover un animal de potrero/corral
1. Ir a **Movimiento**.
2. Digitar código.
3. Elegir destino.
4. Registrar fecha y motivo.
5. Guardar.

### Registrar compra o venta
1. Ir a **Transacción**.
2. Elegir tipo.
3. Ingresar código animal si aplica.
4. Registrar cantidades y valor.
5. Guardar.

### Registrar evento reproductivo
1. Ir a **Reproducción**.
2. Elegir monta, chequeo o parto.
3. Registrar hembra, macho y cría si existe.
4. Guardar.

### Registrar un insumo
1. Ir a **Insumo**.
2. Registrar nombre, tipo, unidad, stock y mínimo.
3. Guardar.

## 3. Trabajo sin internet
- Si no hay señal, la app deja el registro en cola.
- Al volver la conexión, sincroniza automáticamente.
- No cerrar sesión innecesariamente cuando esté trabajando en campo.

## 4. Dashboard
- Muestra stock bajo, ventas semanales, costos y distribución del hato.
- Sirve como lectura ejecutiva rápida desde el celular.

## 5. Descarga de reportes
- En el dashboard tocar **PDF** o **Excel**.
- El archivo se descarga listo para compartir por WhatsApp o correo.

## 6. Reglas operativas recomendadas
- Un solo código por animal. No reutilizar aretes.
- Registrar el movimiento el mismo día. No después.
- Registrar pesos en fechas consistentes para comparar desempeño.
- Registrar vacunas y tratamientos en el momento exacto.
- Revisar alertas de inventario al menos dos veces por semana.
